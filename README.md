# 724 Events


## Description
Déboggage de l'application est le site d'une agence evenementielle.

Objectifs:
- Lister et réparer les bugs en s'appuyant sur React DevTools et JEST
- Rédiger un cahier de recette

## Pre-requis
- NodeJS  >= v16.14.1

## Installation 
- `yarn install`
- `npm install`

## Lancement de l'application
- `yarn start`
- `npm start`

## Tests
- `yarn test`
- `npm test`
